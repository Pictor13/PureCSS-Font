How to Use

Download text-to-css.min.js

Call the text converter with your string of text

var yourVariable = textToCSSFont(yourString);

Optional: add styles.

var yourVariable = textToCSSFont(yourString, {
     bold: true,
     italic: true
     underlined: true
});

Insert on page wherever you'd like.

yourElement.innerHTML = yourVariable;
